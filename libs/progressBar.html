The Progress Bar Library

The progress bar library simply creates and manages progress bars

Functions:

progressBar.new(nX,nY,len,height,nFG,nBG,sName,[nFilled],[tOutput])
-> This creates a new progress bar, at the specified cordinates with len determining the length and height determining the height if set nfilled and toutput are not required

 progressBar.update(sName,nFill)
-> This updates a progress bar and fills it in to the desired amount.

 progressBar.getFill(sName)
-> This gets the current fill amount of a particular progress bar.

 progressBar.calcFill(nLength,nFill)
-> This simply does some math to calculate how many pixels should be filled in to fill up a progress bar.

 progressBar.updateStep(sName,nStep,nTotal)
-> This updates the progress bar, but supports just inputting steps such as '5/7', would be progressBar.updateStep("example",5,7)
